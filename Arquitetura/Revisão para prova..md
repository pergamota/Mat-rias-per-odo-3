
## O que é um computador:

Uma máquina que coleta dados, processa informações, armazena dados e gera resultados nas execuções. 

### O que é automação:

É automatizar algum tipo de trabalho para diminuir a intervenção humana, que inclusive, a principal máquina de automação é o computador.

## Processamento de dados:

É a transformação de dados brutos em dados úteis. Exemplo: transações bancárias (que é um processamento em tempo real).

## Arquitetura e organização:

Arquitetura: é algo que não é físico, é algo que o programador pode ver e serve para projetar, estruturar algo não físico, ou seja, o comportamento funcional de algo.

Organização: é a parte física, ou seja, a estrutura interna de algo, por exemplo.


## O que caracteriza a geração zero:

Eram os computadores mecânicos. Eram usados por analógicos e tinham baixa velocidade e resolvia poucos problemas. 

### Primeira geração:

Computadores passaram a usar a tecnologia de válvulas eletrônicas, fazendo cálculos muito mais rápidos e utilizavam cartões furados para programar. Tinham baixa confiabilidade porque gerava muito calor e as válvulas queimavam.

### Bit:

É a menor unidade de informação. Por exemplo: 1.

### Byte:

É composto por 8 bits, representando um caractere.

## Porque computadores usam o sistema binário:

É a forma mais fácil de saber os estados físicos de um circuito, como se fosse uma booleana, quando é 1 é porque está passando corrente e quando é 0 é porque não tem corrente.


## Von Neumann:  

Ele propôs um sistema computacional com: 

* Uma memória física: para armazenar programas e dados representados por 0 e 1.
* Uma unidade de controle (UC): a função era buscar um programa na memória, instrução por instrução para executa-lo.
* Uma unidade aritmetica e lógica (ULA): a função era executar operações indicadas pelas instruções do programa. 
  
A junção entre UC, ULA e registradores específicos formam a CPU.



### Fases dos computadores:

No começo eram computadores analógicos, depois evoluímos para os transistores que reduziram muito o tamanho do computador porque era mais compacto. Passamos para circuitos integrados que era muito mais rápido e encapsulado. Evoluímos para chips de processadores e agora, microprocessadores.
